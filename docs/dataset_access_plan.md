# Dataset access plan

## Purpose

This document summarises candidate datasets and resources that could support a staged programme of component annotation, imaging replication and molecular convergence analysis.

It is a planning document only. Inclusion of a dataset here does not imply that access has already been granted or that analyses have been performed.

## Dataset categories

The programme requires four broad classes of resource:

1. genetic association and fine-mapping resources;
2. independent imaging and clinical cohorts;
3. transcriptomic, epigenomic and proteomic resources;
4. open annotation databases.

## Candidate resources

| Resource | Intended use | Access status |
|---|---|---|
| UK Biobank | Original context and potential longitudinal imaging extension, subject to existing permissions | Controlled-access |
| ADNI | Alzheimer’s disease imaging, clinical and biomarker replication | Access application required |
| PPMI | Parkinson’s disease imaging, clinical, biomarker and genetic replication | Access application required |
| AMP-PD | Parkinson’s disease genomics, transcriptomics and harmonised clinical context | Access application required |
| AD Knowledge Portal | Alzheimer’s disease transcriptomic, epigenomic and proteomic resources | Registration and dataset-specific permissions may be required |
| NIAGADS / ADSP | Alzheimer’s disease genetics and fine-mapping context | Controlled-access application required |
| GTEx | Tissue-level expression and QTL annotation | Public and controlled-access components |
| GWAS Catalog | Published disease and trait association context | Public |
| Open Targets Genetics | Variant-to-gene and disease-association context | Public |
| Ensembl / VEP | Variant and gene annotation | Public |

## Public repository boundary

This repository will not contain individual-level data from any controlled-access resource.

Where examples are needed, they will use synthetic data or openly shareable summary resources.

Any controlled-access analysis should be documented separately and only shared according to the relevant data-use agreements.

## Initial access priorities

The first access priorities are:

1. ADNI, for Alzheimer’s disease imaging and clinical replication;
2. PPMI, for Parkinson’s disease imaging and clinical replication;
3. AD Knowledge Portal, for molecular annotation and Alzheimer’s disease omics context;
4. AMP-PD, for Parkinson’s disease molecular and clinical context.

## Notes

The immediate purpose of this repository is to document the public protocol and reproducible workflow design.

Empirical analyses using controlled-access datasets will require separate governance, storage, compute and reporting procedures.
